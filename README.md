# RecipeBox-Doc

## Planificarea Aplicației
Acest proiect reprezintă documentația pentru aplicația desktop de gestionare a rețetelor culinare.

### Structura Ecranelor:
* **Pagina principală**: Conține lista de rețete (ex: Pâine casnică, Paste Carbonara) și butonul de adăugare.
* **Detalii rețetă**: Afișează ingredientele necesare și instrucțiunile de preparare.
* **Formular Adăugare**: Permite introducerea titlului pentru o rețetă nouă.
